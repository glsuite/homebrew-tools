## Install

```bash
brew tap glsuite/tools
```

## Formulae

| Formula | Description | Install |
|---------|-------------|---------|
| [gltrace](https://github.com/glsuite/gltrace) | Terminal-first GitLab pipeline log explorer | `brew install gltrace` |

## Usage

```bash
# Add the tap once
brew tap glsuite/tools

# Install
brew install gltrace

# Upgrade
brew upgrade gltrace

# Uninstall
brew uninstall gltrace
brew untap glsuite/tools
```
